Здравствуйте,
Данная программа является моим проектом для Яндекс Лицея.
Справочник для туриста является полезным источником данных для всех, кто хочет и любит путешествовать.
В моем приложении можно найти оптимальную страну для отдыха, узнать необходимую информацию про нее или прочитать важные советы туристам.

Весь код программы был написан на Python с использованием PyQt5 и базы данных SQLLite.

Функции:
Поиск нужной страны по названию
Поиск оптимальной страны для туризма с помощью фильтров
Рейтинги стран по посещаемости, безопасности и т.п
Вкладка с полезными советами для туристов
Особенности:
Информация о странах будет храниться в базе данных 
Таблица стран, которую можно сортировать по критериям
Структура:
Основная программа в Main.py
Данные хранятся в базе данных tourism.sqllite. В ней находятся две таблицы: countries(основные данные о странах), information(остальные данные)
Интерфейс хранится в tourismhelp.ui, starterscreen.ui, 	country_info.ui, advisor.ui
Флаги стран хранятся в папке flags
